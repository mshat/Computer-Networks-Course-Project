# Цель работы
Разработка многопользовательской игры в жанре платформер

# Задание
Разработать многопользовательскую игру с клиент-серверной архитекурой. В качестве протокола прикладного уровня при передаче данных должен использоваться собственный протокол. Приложение должно поддерживать одновременную работу до пяти пользователей. 

Для выполнения поставленной задачи необходимо реализовать программный комплекс, состоящий из следующих компонентов: игровой клиент; сервер для координации состояния подключенных игроков; протокол для обмена информацией между клиентами и сервером с поддержкой сериализации сообщений, передаваемых между клиентом и сервером. В самой игре необходимо реализовать не менее трёх игровых уровней; анимированных игровых персонажей; анимированных монстров. Персонажи должны иметь возможность перемещаться по уровню бегом и прыжками, ударами наносить урон монстрам. Так же должен быт реализован хотя бы один вид взаимодействия персонажа с уровнем (например, нажатие кнопки, переключение рычага и т.д.).
